# <a name="section1">Summary

Simple wiki to write in markdown

# <a name="section2">Dependencies

- ruby >= 1.9.3-p448
- gem >= 1.8.23
- rails >= 4.0.1

# <a name="section3">Getting Started
1. Download

        git clone https://github.com/nemonium/mdwiki.git

2. Gem package install

        cd mdwiki
        bundle install

3. Setup database

        rake db:setup

4. Boot

        rails s
